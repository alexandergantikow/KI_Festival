Eis-Bewertungs-App

Mini-Workshop

Wir entwickeln eine KI-Anwendung Schritt für Schritt

In diesem Mini-Workshop baust du deine eigene kleine App, mit der du verschiedene Eissorten bewerten kannst. Gehe die Aufgaben Schritt für Schritt durch und probiere aus.

Hilfe benötigt? Wenn du Hilfe brauchst, frage gerne die freundlichen Kollegen am Stand.

Einführung: Variablen in der Programmiersprache Python

Variablen sind wie kleine Boxen, in denen du Informationen speichern kannst. Du gibst ihnen einen Namen (dieser steht vor dem =) und kannst ihnen Werte zuweisen (diese stehen nach dem =), zum Beispiel Zahlen oder Texte. Die Variablen merken sich alles, was du ihnen sagst und können das später wieder verwenden.

Zum Beispiel:
name = "Anna" punkte = 5

Aufgabe 1: Titel und Beschreibung

Gib deiner App einen passenden Titel und eine kurze Beschreibung.

Suche im Code die Zeile mit st.title() und trage dort einen Titel ein, zum Beispiel:

st.title("Eis-Bewertungs-App")

Schreibe darunter eine kurze Beschreibung mitst.write(), zum Beispiel:

st.write("Trage deine Lieblings-Eissorte ein und bewerte sie auf einer Skala von 1 bis 10!")

Achtung! Du musst jede Änderung im Code speichern. Das macht man mit dem Tastaturbefehl strg + S.

Aufgabe 2: Eissorte und Bewertung

Baue das Eingabefeld und den Slider für die Bewertung ein:

Erstelle ein Textfeld, in das du eine Eissorte eintragen kannst:

eissorte = st.text_input("Welche Eissorte möchtest du bewerten?")

Füge einen Slider hinzu, mit dem du eine Bewertung von 1 bis 10 auswählen kannst:

bewertung = st.slider("Wie lecker findest du diese Sorte?", min_value=1, max_value=10)

Aufgabe 3: Bewertung anzeigen

Einführung: f-strings in der Programmiersprache Python

Mit sogenannten f-strings kannst du Variablen ganz einfach in Texte einbauen. Das macht deinen Code einfacher zu lesen. Du schreibst einfach ein f vor die Anführungszeichen und setzt den Namen der Variable in geschweifte Klammern:

name = "Anna" punkte = 5 print(f"{name} hat {punkte} Punkte!")

Das Ergebnis ist: Anna hat 5 Punkte!

Füge jetzt noch einen Button hinzu, der - wenn man darauf klickt - die Bewertung der eingegebenen Eissorte anzeigt.

Beispiel:

if st.button("Bewertung abgeben"): st.success(f"Du hast '{eissorte}' mit {bewertung}/10 Punkten bewertet!")

Geschafft! Du hast erfolgreich deine erste Eis-Bewertungs-App programmiert.
